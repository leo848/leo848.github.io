Top 10 Codeeditoren in 2020

Code Editoren 2020

10. UltraEdit

UltraEdit ist dafür bekannt, extrem große Textdateien (die größer sind als 1 GB) extrem schnell öffnen kann. Leider gibt es ansonsten nicht sehr viele Features. Desweiteren ist UltraEdit nicht kostenlos, sondern kostet ca. 80 € pro Jahr inklusive Cloud-Speicher.
UltraEdit für 30 Tage kostenlos testen

9. BBedit

BBedit ist ein Texteditor, der nur für macOS verfügbar ist. Er ist aber ebenfalls wie UltraEdit sehr schnell. Ein gutes Feature von BBedit ist, dass man automatische Dateivorlagen erstellen kann. Außerdem ist BBedit nicht kostenlos, sondern kostet einmal 40 $.
BBedit kostenlos für 30 Tage testen

8. Vim

Vim ist ein multifunktionaler Editor, der sehr schnell und sehr erweiterbar ist. Das wirkt sich leider negativ auf die Nutzerfreundlichkeit aus. Für Anfänger ist Vim daher eher nicht zu empfehlen, für Profis sehr. Wenn man den Editor "beherrscht", kann man sehr schnell arbeiten und muss fast gar nicht die Maus benutzen.
Vim kostenlos herunterladen

7. Notepad++

Notepad++ ist ein Editor, der von Anfang an sehr viele Programmiersprachen unterstützt. Allerdings sieht er auch von Anfang an ziemlich altmodisch aus und man kann nicht viel selbst verändern. Ein weiterer Nachteil ist, dass es nur für Windows verfügbar ist. Deshalb: Platz 7!
Notepad++ kostenlos herunterladen

6. Coda 2

Coda 2 ist ein Editor, der nur für Mac verfügbar ist. Er hat eine schöne UI und einen eigenen Erweiterungsstore. Toll ist auch das Feature, das man die Dateien sofort mit einem iPad synchronisieren kann.
Coda kaufen / kostenlose Version testen

5. Espresso

Espresso ist auch nur für Mac erhältlich. Am besten eignet sich dieser kostenlose Editor für das Bearbeiten von CSS. Außerdem gibt es ein Drag-and-Drop-Feature.
Espresso kostenlos herunterladen

4. Brackets

Dieser kostenlose Editor von Adobe ist vor allem für Webentwicklung gut geeignet. Ein tolles Feature ist die Live-Anzeige von HTML. Beispielsweise können auch Photoshop-Dateien (.psd) in CSS umgewandelt werden, was für Webdesigner ein tolles Feature ist. Außerdem ist der Editor open-source.
Brackets kostenlos herunterladen

3. Sublime Text

Sublime Text ist ein Editor, der sich aufs Programmieren fokussiert und dabei auch viele Möglichkeiten und Tastaturshortcuts bietet. Leider gibt es einige Features, die $80 kosten.
Sublime Text testen

2. Atom

Atom ist ein kostenloser Editor von GitHub. Er ist sehr gut für Webentwicklung und Versionsverwaltung mit Git geeignet. Das Design und die UI sind toll. Da Atom mit Electron erstellt wurde, ist leider die Performance recht schlecht und der Speicherplatzbedarf recht hoch. Es dauert vereinzelt mehr als eine Minute bis zum Start.
Atom kostenlos herunterladen

1. Visual Studio Code

Der kostenlose Texteditor von Microsoft ist eigentlich für so gut wie jede Programmiersprache geeignet. Im Extensions Marketplace findet man für so gut wie alles eine Erweiterung und für fast jede Sprache einen Syntax-Highlighter. Die Keyboard-Shortcuts sind einfach fantastisch und in den Einstellungen ist so gut wie alles verstellbar. Es gibt ein vorinstalliertes Terminal und Versionsverwaltung mit Git. Obwohl auch VS Code auf Electron basiert, started die App meist in unter 15 Sekunden. Ich benutze diesen Editor seit mehr als einem Jahr und habe noch nie darüber nachgedacht, zu wechseln.
Visual Studio Code kostenlos herunterladen

Fira Code zu Visual Studio Code hinzufügen

Schriftart Visual Studio Code Customization

1. Schritt:

Lade dir die zip-Datei, in der die Schriftart enthalten ist, unter diesem Link herunter:

https://fonts.google.com/download?family=Fira%20Code

2. Schritt:

Wähle ein Verzeichnis deiner Wahl und drücke auf Speichern. (Dieses Bild zeigt den Prozess im Browser Chrome. Für andere Browser kann es anders aussehen.) Ich wähle hier das Verzeichnis Downloads/Fira Code.

3. Schritt:

Öffne die Datei im Explorer. Dies funktioniert in Chrome durch Klicken auf den Pfeil nach oben und dann auf "In Ordner anzeigen". Sollte dies nicht funktionieren, öffnet einfach euren Explorer und gebt die Adresse ein, unter der ihr die Datei gespeichert habt.

4. Schritt:

Doppelklicke auf "Fira_Code.zip", dann auf "FiraCode-VariableFont_wght.ttf"

5. Schritt:

Klicke in diesem Fenster auf "Installieren". Wenn die Schriftart fertig installiert ist, kannst du alle Fenster schließen.

6. Schritt:

Öffne nun Visual Studio Code. Drücke dann Strg + ,, um die Einstellungen zu öffnen.

7. Schritt:

Gebe in die obere Suchleiste das folgende ein:
Editor: Font Family

8. Schritt:

Gib in den obersten Eintrag Fira Code ein.


9. Schritt:

Fira Code sollte nun die Standardschriftart in Visual Studio Code sein. Teste es einfach, indem du eine neue Datei erstellst. Aber die Ligaturen werden noch nicht funktionieren. Dazu musst du wieder in die Suchleiste bei den Einstellungen das Folgende eingeben:
Editor: Font Ligatures

10. Schritt:

Klicke auf Edit in settings.json. Dein Cursor wird sich in einer Zeile befinden, in der etwas ähnliches wie dieses steht:

Ersetze hier das null durch true. Jetzt teste es erneut. Es sollte funktionieren!
